# Variables
Variables
# 1. Enteros (int)

Representan números sin decimales, positivos o negativos.

Ejemplo:

edad = 25

# 2. Flotantes (float)

Son números que contienen decimales.

Ejemplo:

precio = 19.99

# 3. Cadenas de texto (str)

Son textos encerrados entre comillas simples ' ' o dobles " ".

Ejemplo:

nombre = "Carlos"

# 4. Booleanos (bool)

Solo pueden ser True (verdadero) o False (falso).
Muy usados para condiciones.

Ejemplo:

es_mayor = True

# 5. Listas (list)

Colecciones ordenadas y modificables. Se escriben con corchetes [ ].

Ejemplo:

frutas = ["manzana", "pera", "uva"]

# 6. Tuplas (tuple)

Parecidas a las listas, pero no se pueden modificar (inmutables).
Se escriben con paréntesis ( ).

Ejemplo:

coordenadas = (10, 20)

# 7. Conjuntos (set)

Colecciones sin orden y sin elementos repetidos.
Se escriben con llaves { }.

Ejemplo:

colores = {"rojo", "azul", "verde"}

# 8. Diccionarios (dict)

Son colecciones que almacenan datos en formato clave : valor.
También usan llaves { }.

Ejemplo:

persona = {
    "nombre": "Carlos",
    "edad": 25,
    "profesion": "Desarrollador"
}

# 9. NoneType (None)

Representa ausencia de valor.

Ejemplo:

resultado = None
